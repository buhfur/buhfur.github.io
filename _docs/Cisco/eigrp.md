## **Commands** 

Show eigrp Neighbors 

```bash
sh ip eigrp nei 
```

Show eigrp Neighbors 

```bash
sh ip eigrp int 
```

Check if EIGRP is enabled on interfaces 
```bash
sh ip proto 
```

Check EIGRP routes in routing
```bash
sh ip route eigrp  
```

Look at EIGRP topology
```bash
sh ip route eigrp  
```
